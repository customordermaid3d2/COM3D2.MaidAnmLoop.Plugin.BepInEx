# 살세 설명

- "Apply" 클릭시 해당 장면에서 자동으로 적용 여부 설정  
- "Apply Repeat" 클릭시 매 프레임마다 반복 적용 여부 설정.  
  다른 플러그인이 덮어쓰는것 방지  
- "Mode" 는 모션 반봅 타입 설정  
- "Auto Apply Scene"에서는 자동 적용될 장면들 목록이 뜨고  
  해당 장면 버튼클릭시 키고 끄기 가능
